# System zarządzania energią w budynkach inteligentnych (SZEBI)

### Cel projektu

Celem projektu było stworzenie inteligentnego systemu informatycznego do zarządzania energią w budynkach komercyjnych i użyteczności publicznej. 

### Funkcjonalności
Aktualny system: 
 - optymalizuje zużycie energii,
 - minimalizuje koszty eksploatacyjne,
 - poprawia komfort użytkowników,
 - wspiera zrównoważony rozwój poprzez redukcję śladu węglowego.

Umożliwia on monitorowanie, analizę, prognozowanie i automatyczne sterowanie urządzeniami zużywającymi energię w budynku, zapewniając elastyczność i adaptację do zmieniających się warunków i potrzeb.

### Wykorzystane technologie
 - Java
 - JavaScript
 - Python
 - CSS
 - PostgreSQL

### Uruchomienie programu

Terminal 1:
```
cd .\python_module\src
python -m app.main
```
Terminal 2:
```
mvn spring-boot:run
```
Terminal 3:
```
cd .\frontend
npm start
```

## Intelligent Building Energy Management System (SZEBI)

### Project Goal

The goal of the project was to develop an intelligent IT system for managing energy consumption in commercial and public utility buildings.

### Features

The current system:

-   optimizes energy usage,
    
-   minimizes operational costs,
    
-   improves user comfort,
    
-   supports sustainable development by reducing the carbon footprint.
    

It enables monitoring, analysis, forecasting, and automatic control of energy‑consuming devices within the building, ensuring flexibility and adaptation to changing conditions and needs.

### Technologies Used

-   Java
    
-   JavaScript
    
-   Python
    
-   CSS
    

### Running the Application
Terminal 1:
```
cd .\python_module\src
python -m app.main
```
Terminal 2:
```
mvn spring-boot:run
```
Terminal 3:
```
cd .\frontend
npm start
```
